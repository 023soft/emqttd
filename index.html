Welcome to eMQTT.

eMQTT is a scalable, fault-tolerant and extensible mqtt broker written in Erlang/OTP.

eMQTT requires Erang R15B+

Startup in Five Minutes


$ git clone git://github.com/emqtt/emqtt.git
$ cd emqtt
$ make && make generate
$ cd rel/emqtt
$ ./bin/emqtt console
